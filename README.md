# I am Developing a LMS full stack project to showcase my full stack skills
### I am using MERN stack in this project

# To start with the project
## step 1
  ### fork the repository
## step 2
  ### clone the repository
    ```
    
    git clone <url>
    
    ```
## step 3
  ### to run the frontend 
  ```
    cd frontend
    npm install
    npm run dev
  ```

  ### to run the backend
  ``` 
    cd backend
    npm install
    npm run dev
  ```
